# java-cli-maven-failsafe-jacoco-pmd-jbehave-junit-hello-world

## Description
A POC for maven app using JUnit5
and jbehave framework with pmd,
jacoco, and failsafe plugin.

## Tech stack
- java
- maven
  - junit
  - jacoco
  - pmd
  - failsafe
  - jbehave

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
